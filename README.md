# labdb
Python学習および開発リハビリを目的とした血液検査データ管理システム

## 概要
カメラ撮影した血液検査結果表の画像を解析し、データベース化するツールです。

## 使用技術
- Python 3.14
- NumPy
- OpenCV
- pytesseract
- SQLite
- Tkinter
  
## 実行方法
% python main.py と実行するとファイル指定のウィンドウが立ち上がります。
testdata/sample_01.jpg を選択して実行を押して下さい。
画像の解析が終わるとテーブルを表示するウィンドウが表示されます。
行をダブルクリックすると別ウィンドウでデータが一画面に表示されます。
